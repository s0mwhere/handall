## Features

- 🎲 "Fun and engaging" gameplay
- 💻 real time hand tracking
- 💯 A score system?!?
- ⚔️ A myriad of obstacle types
- 👋 100% hand control, no need for keyboard
- 📸 AAA graphics
- 🎯 exercise while having fun
- ⚙️📈🗿 minimalism UI
